# uicards
A reusable and customisable css cards design for User interfaces

## Description
```
Utility classes used
mt - margin top
ml - margin left
mr - margin right
mb - margin bottom
my - margin y-axis
mx - margin x-axis

pt - padding top
pl - padding left
pr - padding right
pb - padding bottom
py - padding y-axis
px - padding x-axis

```
